# Weather Dashboard

Weather Dashboard is a fullscreen desktop weather application built with Python and Tkinter.  
It retrieves real-time weather data based on user-approved location access and displays it in a clean, simple UI.

> **Note:** This project is not an official Snake Technologies product it is simply for fun :) 
> It is an independent educational and portfolio project.

---

## Features

- Fullscreen Tkinter GUI  
- User-controlled location permission  
- Live weather data retrieval via API  
- Image/logo support (image located alongside `.py` files)  
- Read-only weather display  
- Emergency exit using ESC key  
- Graceful error handling (API & assets)

---

## Technologies Used

- Python 3  
- Tkinter (GUI)  
- Custom Weather API Module (`request_api.py`)  
- Standard Python libraries: `os`, `sys`, `platform`
